Under the hood

Technical notes

This project centers the map on Boston, plots each Red Line station as a marker, requests prediction data on interaction, and calculates the nearest station based on the user’s coordinates.

For a production deployment, move API keys into a safer configuration flow instead of committing them to a public client-side repository.
