DESCRIPTION

    Web application that utilizes HTML, CSS, JavaScript, Massachusetts Bay Transportation Authority API, and Google Maps API integrations. This web app incorporates map markers, geolocation, and real time scheduling. Each Boston MBTA red line station on the map is represented as a marker. Upon clicking on a station marker, an information window appears that displays the associate station's schedule of upcoming trains and train directions. A red polyline connects each station. Initially, the map is centered on South Station. When the user's geolocation is determined, the user's location is represented by a geolocation marker. The map centers on the user's location automatically. Upon selecting the user's geolocation on the map, an information window appears that displays the nearest Boston MBTA red line station, including distance away in miles. A dotted polyline connects the user's geolocation marker to the marker of the nearest Boston MBTA red line station.

API's Utilized:
  • Google Maps
  • Boston MBTA (Massachusetts Bay Transportation Authority)
